Summary
This project implements a Retrieval-Augmented Generation (RAG) pipeline to process, retrieve, and generate responses from PDF documents. The goal is to enable a system that can answer user queries by extracting information from unstructured PDFs, particularly those containing tables and domain-specific content.
‚ö°
Workflow
PDF Table Extraction: Tables are extracted using Azure OCR.
Embedding Generation: The extracted content is embedded to form a vector store that enables semantic search.
Hybrid Retrieval: Combines both dense vector search and sparse retrieval (BM25) to fetch relevant chunks of information.
Rank Fusion: Applies rank fusion methods like Reciprocal Rank Fusion, Linear Combination, and Combination Sum to combine and improve the retrieval results.
Generation: The retrieved context is sent to an LLM to generate a final, context-aware answer.
